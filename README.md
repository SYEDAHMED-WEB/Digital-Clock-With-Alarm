# ⏰ Digital Clock with Alarm

A modern and responsive **Digital Clock with Alarm** developed using **HTML, CSS, and JavaScript**. The application displays the current time and date in real-time while allowing users to set an alarm using a simple and interactive interface.

---

## 🌟 Live Demo

https://syedahmed-web.github.io/Digital-Clock-With-Alarm/

---

## 📌 Features

- 🕒 Real-time Digital Clock
- 📅 Live Date Display
- ⏰ Set Alarm by Hour, Minute, and AM/PM
- 🔔 Alarm Notification
- 🛑 Stop Alarm Button
- 🎨 Modern Glassmorphism UI Design
- 📱 Responsive Layout
- ⚡ Fast and Lightweight
- 💻 Cross-Browser Compatible

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Web Audio API

## 🚀 How to Run the Project

### Method 1 (Recommended)

1. Download or Clone the repository.
2. Open the project folder in **Visual Studio Code**.
3. Install the **Live Server** extension.
4. Right-click on **index.html**.
5. Select **Open with Live Server**.
6. The application will open automatically in your browser.

---

### Method 2

Simply open the **index.html** file in any modern web browser such as:

- Google Chrome
- Microsoft Edge
- Mozilla Firefox

---

## 🎯 How to Use

1. Launch the application.
2. View the current date and time.
3. Select the desired Hour, Minute, and AM/PM.
4. Click **Set Alarm**.
5. When the selected time is reached, the alarm will start.
6. Click **Stop Alarm** to stop the alarm sound.
